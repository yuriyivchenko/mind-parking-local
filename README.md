Парковка для розуму — це локальний інструмент для тимчасового зберігання думок, ідей, планів і внутрішніх хвостів.

Коли думка записана, вона менше тисне на увагу. Її можна не носити постійно в голові, а припаркувати, повернутися до неї пізніше, завершити або відпустити.
